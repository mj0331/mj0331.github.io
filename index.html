Projects
Websites
I built the website for Clash of Comrades, a bi-monthly tournament for players of the game Tooth and Tail, which is currently being developed by Pocketwatch Games. I also help with organising tournaments for Clash of Comrades. The website has a node backend powered by express.js, uses the Jade(now called Pug) templating engine for generating the HTML and does not use any 3rd party CSS frameworks.

Pocketbot
Pocketbot is a powerful chat bot built for Pocketwatch Game's Discord server. From dice rolls to votebans and rewarding users with game keys to command specific 'personas' via it's custom command framework, there isn't much Pocketbot can't be made to do. For this project I worked toghether with other community members to make this bot a reality. I took care of the command framework and the commands under the 'Ryionbot' group
